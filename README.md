# Prescripto

**Prescripto** is a modern, full-featured clinic management platform built with **React + Vite (frontend)**, **NestJS (backend)**, and **PostgreSQL (database)**.  
It provides a seamless and secure solution for clinics and patients to manage appointments, medical records, and billing.

---

## 🚀 Key Features

- **Appointment Booking** – Patients can book, reschedule, or cancel appointments.  
- **Doctor Dashboard** – Doctors can view and manage their schedules and appointments.  
- **Patient Records** – Secure storage and access to medical history.  
- **Billing & Payments** – Streamlined invoicing and payment management.  
- **Secure & Scalable** – Powered by PostgreSQL and NestJS architecture.

## 📦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v14+
- npm or yarn
- PostgreSQL (local or hosted)

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/ahmed-maatty/Prescripto.git
cd Prescripto

# 2. Install frontend dependencies
cd frontend
npm install
# or: yarn install

# 3. Install backend dependencies
cd ../backend
npm install
# or: yarn install
